## Routing notes

- **Aliases match case-insensitive substring.** "art deco" / "Art Deco" / "ARTDECO" / "1920s gatsby" all hit `art-deco`.
- **Multi-vibe combine**: `/design art-deco + bloomberg-terminal` → merge palettes (deco gold + terminal black bg + amber/green data) and typography (Avant Garde display + IBM Plex Mono body).
- **No vibe match**: skill enters improv mode (`vibes/improv.md`) — reasons from the literal words, picks closest 1-2 vibes, hybridizes.
- **Combining with archetype**: archetypes set the *layout shape*, vibes set the *visual style*. They're orthogonal. `/design dashboard + cyberpunk` is a cyberpunk-styled dashboard.
