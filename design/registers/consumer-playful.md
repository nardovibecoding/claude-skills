# consumer-playful — Duolingo/Notion/Figma

Cultural commitment: warm, friendly, playful but not childish. Rounded everything. Vivid accents. Micro-interactions reward.

## Allowed visual moves

- Border-radius 16-24px standard, 32px+ for hero shapes.
- Vivid accent colors (electric blue, mint, rose, sunshine).
- Mascots / character illustrations OK (Duolingo owl, Notion meeple).
- Micro-animations: bounce on success, shake on error, confetti on milestone.
- Emoji in UI labels acceptable (sparingly).
- Inline gifs / lottie animations for empty states.
- Sticker-style achievement badges.
- Soft drop-shadows + subtle layered cards.

## Forbidden moves

- Clinical / sterile vibes.
- ALL CAPS aggression.
- Hairline borders only (need warmth).
- Mono fonts as primary.
- Black-on-white as default (use off-white + dark warm gray).
- "Enterprise-grade" copy.

## Example brands matching this register

- duolingo, notion, figma, miro, lovable, kakao, line, mercari, pinkoi, dcard, miro, expo

## Sample copywriting voice

- "You're on a 12-day streak!"
- "Let's make this together."
- "Oops, that's not quite right — try again?"
- "🎉 You did it!"

Second person. Contractions. Encouragement language. Exclamation OK (sparingly). Emoji allowed.

## Pairing

- Compatible directions: `modern-minimal` (rounded variant), `soft-warm`.
- Compatible archetypes: `chat`, `landing`, `pricing`, `popup`.
- Avoid: `editorial-monocle`, `brutalist`, `tech-utility`.

## Cultural anti-pattern

If the design feels like a B2B SaaS dashboard with rounded corners — drift. Playful means rewarded interactions, not just rounded.
