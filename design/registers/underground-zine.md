# underground-zine — Are.na/risograph

Cultural commitment: hand-set type, glitch, low-fi, asymmetric. Photocopy aesthetic.

## Allowed visual moves

- Hand-set / pasted-up typography. Mixing 3-4 fonts intentionally.
- Risograph color palette (knock-out spot reds, blues, fluorescent greens).
- Halftone textures, scan-line distortion, JPEG artifacts as feature.
- Asymmetric type placement, rotated headers, overprint.
- Hand-drawn rules + arrows.
- Polaroid-style image frames.
- Visible "made on" timestamps, version numbers, room-tone metadata.
- Marquee scrollers, blinking text (in moderation).

## Forbidden moves

- Polished gradients.
- Sans-serif body in default sizes.
- Centered hero with gradient CTA.
- Glassmorphism.
- Stock photography.
- Anything that looks like SaaS B2B.

## Example brands matching this register

- are-na, dispo, lomography, yeezy.com early-era, rauschenberg-archive, low-tech-magazine, frank-archive

## Sample copywriting voice

- "issue 04 // spring '26"
- "FOR INTERNAL USE / DO NOT DISTRIBUTE"
- "shot on disposable, march 14"
- "we don't ship fast we ship right"

Lower-case OK. ALL CAPS in ironic frame OK. Slashes // for separators. No marketing jargon.

## Pairing

- Compatible directions: `brutalist`, `editorial-monocle` (rough variant).
- Compatible archetypes: `landing`, `footer`, `hero`.
- Avoid: `dashboard`, `chart`, `pricing`, `enterprise-trust` register.

## Cultural anti-pattern

If your zine looks too clean — drift. Real zines have register marks visible, off-grid type, and scanned imperfections.
